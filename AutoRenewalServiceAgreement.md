# 出片相机自动续费服务协议

## 版本更新提示
出片相机近期对《自动续费服务协议》进行了更新和优化，主要涉及冻结期的调整和部分条款描述的更新，以使权利义务描述更加清晰，避免歧义。

**更新时间：2025年10月1日**

## 重要须知
为使用出片相机提供的会员自动续费服务（以下简称"本服务"），您应当事先阅读并遵守《出片相机会员服务协议》及本协议。本协议被视为《出片相机会员服务协议》的补充协议，是其不可分割的组成部分，与其构成统一整体。如本协议与《出片相机会员服务协议》存在冲突的，以本协议为准。

除非您已阅读并接受本协议所有条款，否则请不要开通或使用本服务。当您以接受本服务的方式（包括但不限于点击同意、订阅、开通或续费，或您进行与自动续费服务相关的交易、采用任何其他形式的确认操作，统称为：接受服务的形式）进行下一步操作或使用本服务即表示您已阅读并同意受本协议的约束。如果您未满18周岁，请在法定监护人的陪同下阅读本协议。

## 1. 服务说明

### 1.1 服务定义
本服务是在您开通出片相机会员服务（以下简称"会员服务"）的基础上，出于您对于自动续费的需求，为避免您因疏忽或其他原因导致未能及时续费而推出的服务。

### 1.2 iOS平台自动续费
如您通过苹果公司Apple ID开通本服务，且在下一个订阅期前24小时前未主动明确取消本自动续费服务，则苹果公司将在每个订阅期即将到期前24小时，从您开通本自动续费服务时绑定的Apple ID账户（以下统称"账户"）余额中自动代扣下一个订阅期服务费，从而延长该订阅期对应的会员有效期。

### 1.3 Android平台自动续费
如您通过安卓（Android）产品开通本服务，且在会员期内未主动明确取消本自动续费服务，则相关主体将在会员到期日当天和到期后，从您开通本自动续费服务时绑定的自有充值账户、绑定的第三方支付账户（如微信支付、支付宝）等账户（以下统称"账户"）余额中自动代扣下一个会员期服务费，从而延长该订阅期对应的会员有效期。

### 1.4 特殊支付方式
部分会员服务（如手机话费支付、定制版等）由运营商或相关主体根据实际情况自行决定扣费周期，请以实际扣费时间为准。您应关注上述各类账户及可扣款余额状况，如因账户问题或余额不足导致续费失败而引发的风险或导致的损失将由您自行承担。

### 1.5 续费提醒
为便于您了解自动续费情况，出片相机会根据法律规定，在下一个订阅期扣款前5天以推送通知、应用内消息等方式通知您，请您注意接收。此外，第三方支付渠道也可能向您发送扣款前的提醒通知。

### 1.6 重要提醒
提醒您特别注意，基于自动续费服务的特点，自动代扣将按照本协议1.2、1.3、1.4条的约定方式和时间完成，如果您未在代扣之前明确取消自动续费服务，出片相机将根据您的委托向相关主体发出下一个订阅期的代扣指令，原则上对基于该指令扣除的费用将不予退还。

## 2. 服务使用规则

### 2.1 计费周期
出片相机会员服务的计费周期包括按月、按年等扣费方式（以订阅页面设置为准），出片相机扣除用户选择的指定计费周期的自动续费费用后，将款项记入您支付记录，并同时相应延长您的服务有效期。

### 2.2 扣费差错处理
如在扣费过程出现差错，请您务必及时通知出片相机，出片相机将和您密切配合查明原因，各自承担己方过错造成的损失；若因双方各自存在不均等过错造成的损失，由双方按过错的程度承担相应责任；双方共负责任的，由双方均摊责任。

### 2.3 协议变更
出片相机可根据其业务开展或技术升级等情况变更或修改本协议、有关服务内容、服务价格、规则等（以下简称"自动续费规则"），根据互联网行业之特性，出片相机在做出上述变更或修改前，将根据情况有选择的在出片相机相关页面上公示修改的内容，但无法另行做个别通知。如果您不同意本协议的修改，可以在下一个扣费日前取消已经获取的服务并停止使用；如果您以接受服务的形式继续使用出片相机提供的本服务，则视为您已经接受自动续费规则的全部修改（如服务费用发生调整的，则后续代扣的费用将以调整后的服务费用为准）。

### 2.4 服务费用
出片相机对您开通自动续费服务暂不额外收取任何费用，但出片相机有权根据业务需要或市场变化等原因决定是否对自动续费服务进行额外收费，并在相关页面向您进行公示。

## 3. 中止或终止本服务

### 3.1 服务终止情形
出现下列情况之一的，出片相机有权立即终止向您提供本服务，且无须为此向您或任何第三方承担责任，包括但不限于：
1. 经国家行政或司法机关的生效法律文书确认您存在违法或侵权行为，或者出片相机根据自身的判断，认为您的行为涉嫌违反本协议内容或出片相机平台不时公布的使用规则等内容，或涉嫌违反法律法规的规定的；
2. 您的行为干扰了出片相机任何部分或功能的正常运行；
3. 出片相机认为向您提供本服务存在重大风险的；
4. 其他您严重违反本服务约定之情形的。

### 3.2 运营策略调整
出片相机亦可以根据运营策略需要而停止继续向您提供自动续费服务，如有此等情形发生，出片相机会采取公告的形式通知您，您的会员服务期限自通知您的会员服务周期届满之日起终止。

### 3.3 用户主动取消
您有权随时在出片相机平台相关页面选择终止本服务，终止本服务后，出片相机将停止向您提供本服务。自动续费用户退订方法如下：

**（1）苹果应用内支付（IAP支付）：**
- iOS12及以上系统：打开苹果iOS设备"App Store" → 点击右上角苹果账户头像 → 进入"账户" → 点击"订阅" → 选择出片相机，取消订阅即可；
- iOS12以下系统：打开苹果手机"设置" → 进入"iTunes Store 与 App Store" → 点击"Apple ID"，选择"查看Apple ID"，进入"账户设置"页面，点击"订阅"，选择出片相机，取消订阅即可。

**（2）微信自动续费用户：**
打开微信支付 → 点击右上角进入"支付管理" → 点击"扣费服务" → 选择出片相机会员 → 点击"关闭服务"即可。

**（3）出片相机APP内自动续费用户：**
打开出片相机"我的" → 点击"会员中心" → 点击"管理自动续费" → 点击关闭自动续费即可。

**（4）支付宝自动续费用户：**
打开支付宝 → 点击"我的" → 点击"设置" → 点击"支付设置" → 点击"免密支付/自动扣款" → 选择出片相机会员 → 点击"关闭服务"即可。

### 3.4 状态更新延时
自动续费状态更新可能存在显示延时，以各支付平台的自动续费签约状态为准。

### 3.5 冻结期机制
若您未关闭本服务（退订自动续费），且出片相机通过您绑定的账户未能成功扣除下一计费周期费用，则您的会员服务将进入**7天**的会员冻结期，冻结期内您的会员权益将被暂停，出片相机将在会员冻结期内尝试再次进行扣款。如果在会员冻结期内扣款成功，则您的会员服务将解除冻结，自解除冻结之日起计算下一个计费周期；如果在会员冻结期内未能成功扣款，则出片相机将终止您的会员服务。如您需要继续使用会员服务，需重新开通，开通方式和会员服务内容以具体产品页公示为准。
